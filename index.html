What is CurrySwap?

CurrySwap is a high-yield multi-dapp ecosystem built specifically for your CURRY tokens.

First, users participate in the yield farming event and earn CURRY tokens. Then, the Auction Lobby opens where users may stake their CURRY and earn ETH dividends. NFT rewards that carry benefits to holders will be distributed throughout the entire time.

As the CurrySwap ecosystem grows we’re excited to announce the launch of two new tokens. Both BEP20 tokens built on the Binance Smart Chain (BSC) network, the CurryV2 and Chili tokens are the latest tasteful additions to the Curry lineups. Both these tokens will hold similar functionalities to the Curry token, however the CurryV2 tokens can be staked in the Food Vault to earn ETH/BNB rewards while the Chili tokens will work as the main token of a yield aggregator farming program like bunny.

The Curryv2Token

CurryV2 is a Binance Smart Chain based BEP20 standard token. With similar functionality to ERC20 tokens on the Ethereum blockchain, this BEP20 token is also compatible with the Ethereum Virtual Machine blockchain. This ensures that DeFi projects and other blockchains can participate in cross-chain interoperability.

The CurryV2token has the same functionalities as the Curry token. The esteemed Curry token has undergone auditing and is running successfully on the Ethereum ethernet, hence similar functionality has been incorporated to maintain the interoperability.

A Taste Of CurryV2 Features

Minting

Initial minting will be done to create liquidity on the PancakeSwap protocol. PancakeSwap is similar to Uniswap, and is present on the Ethereum mainnet.

Burn

Burn Tokens present in the Team wallet can be burnt using this functionality.

Limited Total Supply

There will be no extra minting apart from the initial minting mentioned above, which will contribute to the price based growth of the token. All initial liquidity will be burnt, similar to what has been done with the Curry token on the Ethereum platform.

Ownership

Ownership will be renounced (revoked) or will be transferred to a Governance based timelock contract.

Governance

Contracts will allow the community to propose, vote, and implement changes. A snapshot based protocol will be implemented to support governance and voting.

Delegate

Delegate votes from the sender to the delegatee. How it works is that a user can delegate to one address at a time, and the number of votes added to the delegatee’s vote count is equivalent to the balance of CurryV2 token in the user’s account. Votes are delegated from the current block and onward, until the sender delegates again, or transfers their CurryV2.

Get Current Votes

Gets the balance of votes for an account as of the current block.

Get Prior Votes

Gets the prior number of votes for an account at a specific block number. The block number passed must be a finalized block or the function will revert.

Chiliswap Yield Aggregator

There is another token which will be introduced to the CurrySwap ecosystem which will also be deployed on the Binance Smart Chain. It is called the Chili token. Chili tokens will also have the same features, but it will work as the main token of a yield aggregator farming program like bunny.

Ownership of Chili tokens will be transferred to the yield aggregator smart contract. Some exciting swap features with high APY value will enhance the credibility of the CurrySwap ecosystem.

Auction Lobby (Weekly BNB-ETH Deposit Program)

The Auction lobby will be a weekly CURRYV2 token release program based on BNB-ETH LP pair deposit amount.

Each week will have a certain allocation of number of CURRYV2 tokens, users will deposit BNB-ETH LP pair and will get the amount back depending on the contribution amount. It is a year long program spanning 52 weeks. Users can participate in Auction Lobby on Monday of each week from Monday: 00:00:00 am to Monday: 11:59:59 PM UTC . Other than this day the deposit will be reverted back.


Each week one card will be activated r that week. Users can able to see their Claimable CURRYV2 after the deposit over and they can claim CURRYV2 tokens any time.

The received amount will be distributed as follows:
  • 15% will be platform reserve
  • 35% will be allocated towards Food vault (365 bonus)
  • 50% will be allocated towards the Liquidity mining program.
